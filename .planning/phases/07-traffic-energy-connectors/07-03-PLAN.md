```python
class BaseConnector(ABC):
    async def upsert_feature(self, source_id: str, domain: str, geometry_wkt: str, properties: dict) -> str
    async def persist(self, observations: list[Observation]) -> None
    async def _update_staleness(self) -> None

@dataclass
class Observation:
    feature_id: str
    domain: str       # "energy" for SMARD
    values: dict      # {"value_kw": float, "source_type": str}
    timestamp: datetime | None = None
    source_id: str | None = None
```

SMARD API (from RESEARCH.md):
```
BASE = "https://www.smard.de/app/chart_data"
# Step 1: GET /{filter_code}/DE/index_quarterhour.json -> {"timestamps": [...]}
# Step 2: GET /{filter_code}/DE/{filter_code}_DE_quarterhour_{timestamp}.json -> {"series": [[unix_ms, value_mw_or_null], ...]}

GENERATION_FILTERS = {
    "solar": 4068, "wind_onshore": 4067, "wind_offshore": 1225,
    "gas": 4071, "lignite": 1223, "hard_coal": 4069,
    "nuclear": 1224, "hydro": 1226, "biomass": 4066,
}
PRICE_FILTER = 4169  # EUR/MWh day-ahead
```

MaStR via open-mastr (from RESEARCH.md):
```python
from open_mastr import Mastr
db = Mastr()
db.download(data=["solar_extended", "wind_extended", "storage_units"])
# DataFrame columns: Breitengrad, Laengengrad, Nettonennleistung (kW),
# Inbetriebnahmedatum, Landkreis, Lage ("Aufdach"|"Freiflaeche")
```

energy_readings existing schema: time, feature_id, value_kw, source_type
SMARD is national data (no feature_id needed for map). Use a synthetic feature_id via upsert_feature with source_id="smard:generation" or persist without feature_id. Per RESEARCH.md: "no point features -- national grid data has no map coordinates". Store with a sentinel feature or skip feature_id.

Decision: SMARD data uses a synthetic "national grid" feature (source_id="smard:national", domain="energy", geometry=POINT(10.45 51.16) center of Germany). This allows the existing persist() energy branch to work unchanged.
