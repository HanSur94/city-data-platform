## Interaction Contract

### Layer toggles
- Toggle on/off: instant — uses MapLibre `layout.visibility` toggle (raster-opacity pattern for tile layers, visibility for GeoJSON layers)
- Freshness error dot shown when connector `last_successful_fetch` > threshold (traffic: 15 min, energy: 30 min, Autobahn: 5 min)

### Map feature clicks
- Traffic station circle click: opens FeaturePopup with flow data + FreshnessIndicator
- Autobahn marker click: opens FeaturePopup with roadwork/closure detail + detour copy
- Energy installation circle click: opens FeaturePopup with installation type, capacity, year

### KPI tile interaction
- Click Traffic tile: sets `activeDomain = "traffic"`, opens traffic detail panel — same pattern as existing `onSelect` prop
- Click Energy tile: sets `activeDomain = "energy"`, opens energy detail panel

### Detail panel charts
- Energy generation mix: stacked area chart (Recharts `AreaChart`), date range picker applies, renewable sources stacked below fossil
- Energy wholesale price: line chart (Recharts `LineChart`), right y-axis, Euro/MWh
- Traffic flow: one line per selected station (Recharts `LineChart`), x-axis = time, y-axis = Kfz/h
- Roadworks list: static list sorted by distance from Aalen city center, each item shows road + description

### Clustering
- MaStR energy layer: cluster at zoom < 11, show cluster circle with count badge; dissolve at zoom >= 11 to individual markers
- MobiData BW stations: cluster at zoom < 12, dissolve at zoom >= 12 — follows GTFS stops pattern

### SMARD poll
- Frontend polls energy KPI endpoint every 15 min (matches SMARD resolution) — uses existing `useKpi` hook polling interval pattern
