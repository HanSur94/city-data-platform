  <action>
    1. Create backend/app/models/air_quality_grid.py with Pydantic models:
       - GridCell(BaseModel): lon, lat, pm25, pm10, no2, o3 (all float | None)
       - SensorReading(BaseModel): feature_id, lon, lat, pm25, pm10, no2, o3 (float | None each)

    2. Create backend/app/connectors/air_quality_grid.py:
       - class AirQualityGridConnector(BaseConnector) -- computation connector
       - Does NOT use standard fetch/normalize/persist pipeline (like BusInterpolationConnector)
       - Override run() to:
         a. Query latest reading per sensor from air_quality_readings joined with features WHERE domain='air_quality' AND town_id=self.town.id. Get lat/lon from ST_Y/ST_X(geometry).
         b. Generate grid points: iterate from bbox.lon_min to bbox.lon_max in steps of ~0.005 (~500m at this latitude), and bbox.lat_min to bbox.lat_max in steps of ~0.0045 (~500m). Use config.get("grid_step_deg", 0.005) for configurability.
         c. For each grid cell, compute IDW interpolation for each pollutant separately:
            - idw(sensors, cell_lon, cell_lat, pollutant, power=2) -> float | None
            - Formula: sum(value_i / dist_i^power) / sum(1 / dist_i^power)
            - Skip sensors with None for that pollutant
            - If dist < 1m, return sensor value directly (avoid div-by-zero)
            - If no sensors have data for that pollutant, return None
            - Use haversine for distance (copy _haversine from bus_interpolation.py)
         d. Upsert each grid cell as a feature:
            - source_id: f"air-grid-{row}-{col}" (deterministic for ON CONFLICT upsert)
            - domain: "air_quality"
            - geometry_wkt: f"POINT({lon} {lat})"
            - properties: {"feature_type": "air_grid", "pm25": val, "pm10": val, "no2": val, "o3": val}
         e. Call self._update_staleness()
       - Do NOT write to air_quality_readings (grid cells are computed, not measured)
       - Log sensor count and grid cell count at INFO level

    3. Create backend/tests/connectors/test_air_quality_grid.py:
       - Extract IDW function as a standalone pure function `idw_interpolate(sensors, lon, lat, pollutant, power=2)` for testability
       - Extract grid generation as `generate_grid(bbox, step_deg)` -> list of (lon, lat, row, col)
       - Test the pure functions directly without database
