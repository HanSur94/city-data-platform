## Interaction Contracts

### Layer Toggle

- Default state: both Transit and Air Quality layers ON
- Toggle OFF: removes MapLibre layer from map immediately (no reload, no fade)
- Toggle ON: adds layer back; triggers data refetch if data is >5 minutes stale
- Layer state stored in React useState; no URL encoding in Phase 4 (deferred to Phase 5)

### Map Feature Click / Hover

Trigger: click on a stop dot, route line, or air quality sensor point.

Popup behavior:
- Renders as `FeaturePopup` component at click coordinates
- Max-width: 200px (per CONTEXT.md)
- Closes on map click outside popup or on Escape key
- One popup open at a time — opening a new one closes the previous

Popup content contract by domain:

**Transit stop popup:**
```
[Stop name]              ← Heading 16px/600
Line: [route short name] ← Body 14px/400
Type: Bus / Train / Tram ← Label 12px/400
[FreshnessIndicator] Updated [relative time]
```

**Air quality sensor popup:**
```
[Sensor name / location] ← Heading 16px/600
PM2.5: [value] µg/m³    ← Body 14px/400
PM10:  [value] µg/m³    ← Body 14px/400
NO₂:   [value] µg/m³    ← Body 14px/400
[AQI Badge: tier label]  ← shadcn Badge with tier hex bg
[FreshnessIndicator] Updated [relative time]
```

### Freshness Timestamp

- Display format: relative time — "2 min ago", "28 min ago", "1 h ago"
- Do NOT display absolute ISO timestamps in the popup body
- FreshnessIndicator dot: green if < 5 min, yellow if 5–30 min, red if > 30 min
- Tooltip on hover shows exact timestamp: "Last updated: 14:32:07"

### Clustering (Transit Stops)

- Zoom < 14: cluster stops into circle markers showing count
- Zoom ≥ 14: dissolve to individual stop dots
- Cluster circle: 24px diameter, secondary color background (#f4f5f7), count label 12px/400
- Individual stop dot: 8px diameter, route-type color fill, white 1px stroke

### AQI Heatmap

- Rendered as MapLibre heatmap layer, not individual point markers
- Interpolation radius: 40px at zoom 12, scales with zoom
- Intensity: 0.8 (not fully opaque — base map visible beneath)
- Color ramp: mirrors AQI_TIERS — #00c853 → #ffeb3b → #ff9800 → #f44336 → #b71c1c
- When heatmap layer is active, clicking near a sensor point still opens the point popup (heatmap is visual-only; point data drives popup)

### Data Refresh

- Frontend polls `/api/layers/{domain}?town=aalen` on mount and every 60 seconds
- No WebSocket, no SSE in Phase 4 (out of scope per REQUIREMENTS.md)
- During refetch: existing layer data remains visible (no loading flash)
- On fetch error: FreshnessIndicator turns red; no error modal
