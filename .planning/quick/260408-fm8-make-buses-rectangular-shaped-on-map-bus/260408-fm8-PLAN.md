**Step 1 — Register bus-rect image on map load:**

Add a useEffect that runs once when mapInstance becomes available. Use `map.addImage()` to register a custom 'bus-rect' image drawn on an HTMLCanvasElement:

```typescript
useEffect(() => {
  const map = mapInstance?.getMap();
  if (!map) return;

  // Draw a 20x10 rectangle (bus shape) on a canvas
  const size = { w: 24, h: 12 };
  const canvas = document.createElement('canvas');
  canvas.width = size.w;
  canvas.height = size.h;
  const ctx = canvas.getContext('2d')!;

  // Rounded rectangle body (filled with white, stroke added per-feature via icon-color)
  const r = 3;
  ctx.beginPath();
  ctx.moveTo(r, 0);
  ctx.lineTo(size.w - r, 0);
  ctx.quadraticCurveTo(size.w, 0, size.w, r);
  ctx.lineTo(size.w, size.h - r);
  ctx.quadraticCurveTo(size.w, size.h, size.w - r, size.h);
  ctx.lineTo(r, size.h);
  ctx.quadraticCurveTo(0, size.h, 0, size.h - r);
  ctx.lineTo(0, r);
  ctx.quadraticCurveTo(0, 0, r, 0);
  ctx.closePath();
  ctx.fillStyle = '#ffffff';
  ctx.fill();

  map.addImage('bus-rect', canvas, { sdf: true });
  // SDF mode allows icon-color to tint the white shape with per-feature colors
}, [mapInstance]);
```

Note: `sdf: true` enables icon-color tinting from a white source image — this is how MapLibre allows data-driven icon colors on a single shared image.
